# Agenda de Contatos 
Este projeto salva os contatos em um arquivo. As informações gravadas são:
- Nome.
- Telefone.
- Email.

O arquivo `contatos.txt` é gravado na raiz da aplicação.

## Menu principal (opções)
#### 1 - Adicionar contato
#### 2 - Remover contato
#### 3 - Buscar contato
#### 4 - Listar contato
#### 0 - Sair
